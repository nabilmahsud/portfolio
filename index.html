01. About Me

Hi, I'm Nabil. I've spent the last three years writing software across TypeScript, Python, and Java in three engineering roles, building APIs in Nest.js and FastAPI, shipping React and React Native interfaces, designing PostgreSQL data layers, and putting Docker and CI/CD around the whole thing.
